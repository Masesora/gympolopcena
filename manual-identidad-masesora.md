## 2. LOGOTIPO

### 2.1 Estructura del logotipo

El logotipo de MASESORA tiene dos componentes que pueden usarse juntos o por separado:

- **Símbolo (icono)**: ola dorada ascendente con punto final, sobre fondo navy o transparente. Representa el recorrido del cliente del caos al control. Se usa como icono independiente (favicon, redes sociales, sello documental).
- **Wordmark (logo principal)**: "MASESORA" en Montserrat ExtraBold + "DINAMISMO EMPRESARIAL" en Montserrat Medium debajo, con línea dorada de separación. Sin la ola.

El **wordmark y el icono nunca se solapan** — se usan en contextos distintos.

### 2.2 Variantes oficiales

Todas almacenadas en `assets/`:

| Archivo | Uso |
|---|---|
| `masesora-logo-icon.svg` | Solo símbolo · cuadrado navy con ola dorada |
| `masesora-logo-icon-light.svg` | Solo símbolo · fondo blanco con ola dorada |
| `masesora-icon-transparent.svg` | Solo símbolo sin fondo (icono genérico) |
| `masesora-wordmark-transparent.svg` | Wordmark sin fondo · MASESORA en `currentColor` + tagline dorado |
| `masesora-logo-light.svg` | Wordmark sobre fondo blanco · MASESORA navy + tagline dorado |

| `favicon.svg` | Icono navegador · 32×32 px (basado en la ola dorada) |

### 2.3 Área de reserva

Mantener un espacio vacío alrededor del logo equivalente a **la altura de la letra "M"** del wordmark. Ningún elemento puede invadir esa zona.

## 4. TIPOGRAFÍA CORPORATIVA

### 4.1 Tipografía principal — Montserrat

**Montserrat** es la tipografía geométrica sans-serif oficial de MASESORA. Gratuita, de Google Fonts, con presencia institucional limpia.

> Fuente: https://fonts.google.com/specimen/Montserrat

### 4.2 Aplicación universal

**Toda la marca MASESORA usa exclusivamente Montserrat.** Sin excepciones. Logo, web, papelería, presentaciones, firma email — todo en Montserrat.

**Excepción premium**: *Cormorant Garamond Italic* puede usarse únicamente como tipografía de apoyo para el eslogan en **diapositivas de cierre** y **papelería física premium** (tarjeta de visita reverso, papel carta). En ningún otro contexto ni pantalla web.

### 4.3 Jerarquía tipográfica

| Uso | Tipografía | Estilo | Tracking |
|---|---|---|---|
| **Wordmark logo (MASESORA)** | Montserrat ExtraBold (800) | All Caps | +100 |
| **Tagline (DINAMISMO EMPRESARIAL)** | Montserrat Medium (500) | All Caps | +420 |
| **H1 hero** | Montserrat Bold (700) | All Caps | +30 |
| **H2 sección** | Montserrat Bold (700) | All Caps | +20 |
| **H3 subsección** | Montserrat SemiBold (600) | Mixed Case | +10 |
| **Eslogan / Eyebrow** | Montserrat Medium (500) | All Caps | +200 / +300 |
| **Cuerpo de texto** | Montserrat Regular (400) | Sentence case | 0 |
| **Cita / Testimonio** | Montserrat Light Italic (300) | Sentence case | 0 |
| **Datos / KPIs** | Montserrat Bold (700) | Tabular nums | 0 |

### 4.4 Tamaños base (responsive web)

## 5. SISTEMA GRÁFICO

### 5.1 Patrón corporativo — Interconexión geométrica

**Concepto**: líneas finas y formas geométricas que se entrelazan o superponen ligeramente.

**Significado**: la red de soluciones, la estructura clínica y el dinamismo empresarial.

**Aplicación**:
- Opacidad: **5% – 15%** (nunca dominante)
- Color: Azul Marino al 8% sobre fondos claros · Dorado al 12% sobre fondos oscuros
- Uso: fondos de hero, papelería, reverso de presentaciones
- **Nunca debe competir con el contenido**: siempre actúa como soporte sutil

### 5.2 Símbolo "Ola Dorada"

El símbolo del logo (ola creciente con punto final) representa:
- Onda de cambio operativo
- Dinamismo empresarial
- Recorrido del cliente del caos al control

Puede usarse aislado como icono pequeño en presentaciones, documentos clínicos (PRD, Certificado de Alta) y firmas de email.

## 6. APLICACIÓN DIGITAL — www.masesora.com

### 6.1 Estructura visual obligatoria

| Elemento | Especificación |
|---|---|
| **Header (NAV)** | Fondo Azul Marino `#0F1A35` · Logo variante navy · Links blanco rgba(.7) / hover dorado |
| **Cuerpo principal** | Fondo Blanco `#FFFFFF` · Texto Azul Marino · Acentos dorados |
| **Botón CTA primario** | Fondo Dorado `#B89D52` · Texto Azul Marino `#0F1A35` · Hover Dorado Claro `#D4B96A` |
| **Botón secundario** | Fondo transparente · Borde Azul Marino al 30% · Texto Azul Marino · Hover borde dorado |
| **Bloques cesura** | (MAS@FRAME, Manifiesto) Fondo Azul Marino · Texto blanco · Acentos dorados |
| **Footer** | Fondo Azul Marino · Texto blanco rgba(.6) · Links dorados en hover |
| **Patrón de fondo** | Aplicado en hero al 5–8% de opacidad |

### 6.2 Arquitectura visual recomendada

```
NAV               → Azul Marino   (encuadre superior)
HERO              → Blanco        (con patrón sutil 5-8%)
PRINCIPIOS        → Blanco        
TESTIMONIOS       → Blanco        (cards azul claro o cream)
BIO               → Blanco        
1er SISTEMA       → Azul Marino   (cesura visual MAS@FRAME)
MANIFIESTO        → Azul Marino   (cesura final / cierre dramático)
FOOTER            → Azul Marino   (encuadre inferior)
```

Esta estructura crea ritmo visual: **claro → cesura navy → claro → cesura final**, respetando tanto el manual MASESORA como el manual MAS@FRAME.

### 6.3 Tokens de UI y microinteracciones

| Token | Valor |
|---|---|
| **Border-radius botones** | `999px` (pill) |
| **Border-radius tarjetas** | `14px – 18px` (orgánico) |
| **Hover botón primario** | `translateY(-3px)` + shimmer dorado + brillo |
| **Sombras** | Profundas y dramáticas · solo grises o azules muy diluidos · sin color |

### 6.4 Don'ts — qué NUNCA hacer

- ❌ Usar fondo negro puro `#000000` (siempre Azul Marino `#0F1A35`)
- ❌ Usar dorados distintos al `#B89D52` (no `#C8A84B`, no `#FFD700`, no `#DAA520`)
- ❌ Mezclar Montserrat con Bebas Neue, Fraunces, Playfair u otras tipografías en contextos web
- ❌ Usar Cormorant Garamond en pantalla (exclusivo de papelería física y diapositivas de cierre)
- ❌ Usar el patrón geométrico a más del 15% de opacidad
- ❌ Colocar el wordmark sobre el patrón sin fondo neutro
- ❌ Modificar las proporciones o pesos del wordmark
- ❌ Usar sombras de color (solo grises o azules muy diluidos)
- ❌ Mezclar acentos teal `#0ABFBC` o red `#E83C2A` ajenos a la paleta primaria
- ❌ Aplicar emojis Unicode en lugar de iconos vectoriales SVG
